# 插件系统实现说明



## 实现方案

实现了一个完整的插件化系统，具有以下特性：

### 核心功能

1. **动态加载** - 通过上传ZIP文件即可添加新连接器
2. **REST API** - 提供完整的HTTP接口管理插件
3. **命令行工具** - 便捷的CLI工具
4. **版本管理** - 支持插件版本控制和依赖管理
5. **无需重启** - 安装插件后无需重启网关即可使用

## 使用示例

### 场景：添加OPC UA连接器

```bash
# 步骤1: 打包连接器
python tools/plugin_packager.py \
  thingsboard_gateway/connectors/opcua \
  -o opcua_plugin.zip \
  -t opcua \
  -v 1.0.0

# 步骤2: 安装插件（两种方式任选其一）

# 方式A: 命令行
python tools/plugin_installer.py install opcua_plugin.zip

# 方式B: REST API（上传文件）
curl -X POST http://localhost:9001/api/plugins/upload \
  -F "file=@opcua_plugin.zip"

# 步骤3: 在配置中使用
{
  "connectors": [
    {
      "name": "OPC UA Connector",
      "type": "opcua",
      "configuration": "opcua.json"
    }
  ]
}

# 完成！OPC UA连接器立即可用
```

## 系统架构

```
┌─────────────────────────────────────────────┐
│  1. 用户上传 opcua_plugin.zip               │
└─────────────────┬───────────────────────────┘
                  ↓
┌─────────────────────────────────────────────┐
│  2. PluginManager                            │
│     - 解压ZIP文件                            │
│     - 验证plugin.json                        │
│     - 安装依赖                               │
│     - 复制文件到 config/plugins/opcua/       │
└─────────────────┬───────────────────────────┘
                  ↓
┌─────────────────────────────────────────────┐
│  3. GatewayPluginIntegration                │
│     - 注册插件路径                           │
│     - 添加到TBModuleLoader                   │
└─────────────────┬───────────────────────────┘
                  ↓
┌─────────────────────────────────────────────┐
│  4. TBGatewayService                        │
│     - 加载连接器时优先搜索插件路径            │
│     - 找到并加载OpcuaConnector               │
│     - 开始数据采集                           │
└─────────────────────────────────────────────┘
```

## 快速开始

### 1. 运行示例
```bash
bash examples/QUICK_START.sh
```

### 2. 启用插件系统

在 `config/tb_gateway.json` 中添加：

```json
{
  "plugins": {
    "enabled": true,
    "enable_api": true,
    "api_host": "0.0.0.0",
    "api_port": 9001
  }
}
```

### 3. 管理插件

```bash
# 列出所有插件
python tools/plugin_installer.py list

# 安装插件
python tools/plugin_installer.py install plugin.zip

# 查看插件信息
python tools/plugin_installer.py info plugin_name

# 卸载插件
python tools/plugin_installer.py uninstall plugin_name
```

### 4. 使用REST API

```bash
# 健康检查
curl http://localhost:9001/api/plugins/health

# 列出插件
curl http://localhost:9001/api/plugins

# 上传安装
curl -X POST http://localhost:9001/api/plugins/upload \
  -F "file=@plugin.zip"

# 卸载
curl -X DELETE http://localhost:9001/api/plugins/plugin_name
```

## 插件包结构

```
my_connector_plugin.zip
├── plugin.json              # 必需：插件清单
├── my_connector.py          # 必需：连接器实现
├── my_uplink_converter.py   # 可选：上行转换器
```

plugin.json示例：

```json
{
  "name": "my_connector_plugin",
  "version": "1.0.0",
  "plugin_type": "connector",
  "connector_type": "myprotocol",
  "display_name": "My Protocol Connector",
  "description": "A connector for My Protocol",
  "author": "Your Name",
  "license": "Apache-2.0",
  "python_requires": ">=3.7",
  "dependencies": ["some-package>=1.0.0"],
  "entry_point": "MyConnector",
  "module_name": "my_connector",
  "gateway_version": ">=3.0"
}
```

## 完整流程示例

### 场景：从零开始添加ModBus连接器

```bash
# 1. 打包ModBus连接器
python tools/plugin_packager.py \
  thingsboard_gateway/connectors/modbus \
  -o modbus_plugin.zip \
  -t modbus \
  -v 1.0.0 \
  -d "ModBus connector" \
  -a "ThingsBoard" \
  --deps "pymodbus>=2.5.0"

# 2. 安装插件
python tools/plugin_installer.py install modbus_plugin.zip

# 输出：
# ✓ Plugin modbus_connector_plugin installed successfully

# 3. 验证安装
python tools/plugin_installer.py list

# 输出：
# 名称                          版本      类型           状态
# ----------------------------------------------------------------------
# modbus_connector_plugin       1.0.0     modbus        已启用

# 4. 创建ModBus配置文件 config/modbus.json
{
  "master": {
    "slaves": [
      {
        "host": "192.168.1.100",
        "port": 502,
        "deviceName": "ModBus Device",
        "deviceType": "default"
      }
    ]
  }
}

# 5. 在主配置中添加连接器 config/tb_gateway.json
{
  "connectors": [
    {
      "name": "ModBus Connector",
      "type": "modbus",
      "configuration": "modbus.json"
    }
  ]
}

# 6. 重启网关或重新加载配置
# ModBus连接器立即开始工作！
```

## REST API端点

| 方法 | 端点 | 描述 |
|------|------|------|
| GET | `/api/plugins` | 列出所有插件 |
| GET | `/api/plugins/{name}` | 获取插件详情 |
| POST | `/api/plugins/upload` | 上传并安装插件 |
| DELETE | `/api/plugins/{name}` | 卸载插件 |
| POST | `/api/plugins/{name}/enable` | 启用插件 |
| POST | `/api/plugins/{name}/disable` | 禁用插件 |
| GET | `/api/plugins/health` | 健康检查 |
